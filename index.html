Computer Graphics Project

This was an exam project created during the course 02561 - Computer Graphics at DTU in the autumn of 2021. It features a submarine exploring an infinite underwater landscape. The accompanying report can be found here.

Controls

Use the W/A/S/D and Q/E keys to move the submarine around.

Technology

A world is procedurally generated by applying a marching cube algorithm (1) to a 3D-simplex noise function (2). The project is implemented in JavaScript, and rendered in the browser with WebGL.
